# 煤炉快速下单 Web 示例

这是一个最小可运行的 Web 原型，用于：
1. 输入一个煤炉商品页 URL
2. 填写账号、密码、数量、地址
3. 模拟下单并显示订单结果

## 快速开始

1. 在终端进入项目目录，运行 `npm install`。
2. 运行 `npm start`（默认启动后端服务 http://localhost:3000）。
3. 在浏览器打开 `http://localhost:3000/index.html`。
4. 填写商品页地址、账号、密码、数量，点击“立即下单”。

## 文件说明

- `index.html`：页面结构
- `styles.css`：样式
- `app.js`：下单逻辑（模拟）

## 真实对接建议

本示例当前是前端本地模拟：
- 真实下单需要与后端 API 对接（登录、商品查询、提交订单）
- 账号信息应使用 HTTPS + 服务器端安全存储，前端不要直接保存密码
- 可扩展：下单历史、订单查询、支付结果回调
